## Run Locally

Clone the project

```bash
  git clone ...
```

Go to the project directory

```bash
  cd ...
```

Create env

```bash
  python -m venv .venv
```

Use the env (in Linux)

```bash
  source .venv/bin/activate
```

Use the env (in Windows)

```bash
  .venv\Scripts\activate
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the program

```bash
  python view.py
```
